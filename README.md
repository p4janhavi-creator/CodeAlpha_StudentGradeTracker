# Student Grade Tracker

A console-based Java application that allows users to:
- Add students
- Record scores
- Calculate averages
- Generate summary reports

## Technologies Used
- Java
- ArrayList
- Object-Oriented Programming

## How to Run
1. Compile:
   javac StudentGradeTracker.java
2. Run:
   java StudentGradeTracker
